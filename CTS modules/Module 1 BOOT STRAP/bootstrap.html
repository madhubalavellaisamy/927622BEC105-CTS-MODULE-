Bootstrap Directory Structure

Bootstrap Directory Structure

  • bootstrap.css: Full unminified CSS for customization.
  • bootstrap.min.css: Minified production-ready CSS (used in production for speed).
  • bootstrap.rtl.css: RTL (Right-to-Left) support for languages like Arabic or Hebrew.

  • bootstrap.bundle.min.js: Contains all Bootstrap JS plugins + Popper (used for tooltips, dropdowns, etc.).
  • bootstrap.min.js: Bootstrap JS only (requires separate Popper).
  • bootstrap.esm.js: Module version for advanced bundlers like Webpack or Rollup.

  • bootstrap-icons.css: CSS file to load Bootstrap Icons.
  • fonts/ or svgs/: Contains SVG files or font icons used by Bootstrap Icons.
  • Note: This folder is available only if you download Bootstrap Icons separately from icons.getbootstrap.com.
